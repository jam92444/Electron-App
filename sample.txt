# Small improvements I recommend

## 1. Add build version

Users like seeing versions.

Example:

```json
"version": "1.0.0"
```

in `package.json`.

---

## 2. Add shortcut creation

Inside `nsis`:

```js
nsis: {
  oneClick: false,
  allowElevation: true,
  allowToChangeInstallationDirectory: true,
  createDesktopShortcut: true,
  createStartMenuShortcut: true
}
```

This automatically adds shortcuts.

---

## 3. Add auto updater later (optional)

Electron supports updates using:

* electron-updater

This lets your app update automatically.

Example:

```
Check for updates
Download
Restart
```

Many professional apps use this.

---

## 4. Protect SQLite database location

Instead of shipping DB inside the app folder, store it in:

```
userData folder
```

Electron path:

```js
app.getPath("userData")
```

Example location:

```
C:\Users\User\AppData\Roaming\InviOX OfflineApp\
```

This prevents **data loss during updates**.

---

## 5. Add backup button in app

Since you use **SQLite**, backup is easy:

Just copy:

```
database.db
```

Add UI:

```
Settings → Backup Database
Settings → Restore Backup
```

Shop owners **love this feature**.
